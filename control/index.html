风扇 / 硬盘传感器 需要本地代理

Web 页面无法直接读取风扇转速、硬盘温度、SMART 健康状态。需要 Windows 本地服务或桌面端桥接。

权限边界

如果后续需要真实 CPU 占用率、风扇转速、硬盘温度、SMART、物理磁盘列表，可以增加一个 Windows 本地采集服务，再由此页面通过 localhost API 读取。

系统信息会根据浏览器权限和平台能力显示，部分硬件字段可能不可用。
