# 模块 01：股票学习系统

## 1. 模块目标

帮助新手从零建立股票知识体系，并把学习过程沉淀到网站里。

学习系统不是文章堆砌，而是要形成：

- 知识地图
- 学习卡片
- 每日训练
- 概念测试
- 交易前检查清单
- 错误案例复盘

## 2. 学习路径

建议按以下顺序学习：

```text
市场基础
  -> 交易规则
  -> K线和成交量
  -> 趋势和支撑压力
  -> 基本面分析
  -> 技术分析
  -> 风险控制
  -> 策略和回测
  -> AI 辅助研究
```

## 6. AI 使用方式

AI 可以提供：

- 用新手能懂的话解释概念
- 根据用户学习进度推荐下一个主题
- 生成小测题
- 解释错误答案
- 把真实股票案例转成学习卡片

AI 提示词示例：

```text
我是股票新手，请用 300 字以内解释 RSI。
要求：
1. 不使用复杂公式
2. 说明它能看什么
3. 说明它不能单独决定买卖
4. 给一个常见误区
```

## 7. 验收标准

- 用户能按路径学习基础知识
- 每个概念都有简短解释和常见误区
- 能记录个人笔记
- 能通过小测题检查理解程度
- 能把交易错误沉淀为学习内容
